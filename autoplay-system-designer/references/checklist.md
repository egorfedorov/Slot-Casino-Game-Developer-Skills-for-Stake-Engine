# Autoplay Checklist

## UX

- Auto-play requires confirmation before start.
- Spin button shows Stop while auto-play is running.
- Remaining spins count is visible.

## Stop Conditions

- Manual stop overrides immediately.
- Stop on balance below minimum bet.
- Stop on session invalidation or error.
- Stop on feature trigger if configured.

## Compliance

- Confirmation includes cost multipliers for special modes.
- Social wording avoids restricted terms.
