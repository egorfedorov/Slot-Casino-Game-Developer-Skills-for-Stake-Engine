# Workflow

## 1. Policy Definition

- Define max spins and stop conditions.
- Define balance and error stop rules.

## 2. UI Mapping

- Define Spin/Stop button states.
- Define counters and stop condition display.

## 3. Confirmation Flow

- Define confirmation dialog content.
- Define cost and multiplier disclosure.

## 4. Validation

- Validate stop rules and manual override.
- Validate compliance wording for social jurisdictions.

## 5. Handoff

- Provide policy, UI mapping, and QA plan.
