# Workflow

## 1. Motion Inventory

- Enumerate motion targets by game state (idle, loading, between rounds, win, bonus).
- Record trigger and cancel conditions per state.

## 2. Pattern Grammar

- Define grid size, palette, and state encoding.
- Choose discrete vs continuous motion strategy.

## 3. Recipe Authoring

- Produce CSS keyframes and class usage per recipe.
- Set timing budgets and step counts.

## 4. Validation

- Confirm readability and no overlap with critical HUD.
- Confirm reduced-motion fallback and CPU cost bounds.

## 5. Handoff

- Deliver spec, recipes, and integration points.
- Provide risk notes and sign-off checklist.
