# Pattern Library

## Idle

- `idleSparkGrid`: low-contrast grid shimmer.
- `softPaletteDrift`: slow hue shift for background plates.

## Loading / Bet Lock

- `loadingOrbit`: rotating conic gradient.
- `stripeStepLoop`: stepped stripe motion using `steps()`.

## Between Rounds

- `barcodeSweep`: stepped scan using repeating gradients.
- `quietPulse`: short opacity pulse with scale in/out.

## Win Reveal

- `winAccentScan`: fast scan band for highlight.
- `payoutGlow`: subtle glow ramp with `filter: drop-shadow`.

## Bonus Entry

- `doublePulse`: two short pulses under 1s total.
- `accentBurst`: brief gradient burst + quick settle.
